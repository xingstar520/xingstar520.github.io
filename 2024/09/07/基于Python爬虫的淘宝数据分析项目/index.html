Jie2024-09-072024-09-19

[toc]

一.项目介绍

该项目是基于Python爬虫的淘宝服装数据分析项目，以致于帮助商家了解当前服装市场的需求，制定更加精确的营销策略。首先，需要爬取淘宝中关于服装的大量数据，然后对数据进行处理和分析，比如处理重复值，提取关键信息，统计分析等等，最后，通过Echarts将处理后的数据进行可视化展示，可以更为直观的看到服装市场的趋势和变化。

二.爬虫代码

代码分析

导入模块

模块作用
json用于处理 JSON 数据。
os用于操作系统相关功能，如文件路径操作。
requests用于发送 HTTP 请求。
selenium 相关模块用于自动化浏览器操作。
pyquery用于解析 HTML 文档。
time用于时间相关操作。
openpyxl用于操作 Excel 文件`

代码首先从主函数进入，传入三个参数，运行TaobaoCrawler爬虫类

初始化方法 __init_：

  1. 初始化爬虫的关键字、起始页和结束页。
  2. 初始化浏览器驱动、等待对象、Excel 文件和工作表。
  3. 创建存储图片的文件夹。

方法 create_images_folder： 检查并创建存储图片的文件夹。
方法 download_image： 下载并保存图片到本地。
方法 init_driver： 初始化并配置 Chrome 浏览器驱动。
方法 init_excel： 初始化 Excel 文件和工作表，并设置表头。

方法 search_goods：

  1. 打开淘宝主页，输入关键字并搜索商品。
  2. 如果起始页不是第一页，跳转到指定页码。
  3. 获取商品信息。

方法 page_turning： 翻页并获取商品信息

方法 get_goods： 解析当前页面的商品信息并保存到 Excel 和 JSON 文件中。

方法 save_excel： 保存 Excel 文件。
方法 save_json： 保存 JSON 文件。
方法 run： 执行爬虫任务，保存数据并关闭浏览器。

三. 数据处理

  1. 导入模块：fs：用于文件系统操作，使用 promises 以便使用异步函数。
    path：用于处理和转换文件路径。
  2. 变量声明：data：用于存储从 JSON 文件加载的数据。
    d：用于存储处理后的数据。
  3. 函数 loadData：
    1. 异步函数，用于加载 JSON 文件中的数据。
    2. 使用 fs.readFile 读取文件内容，并将其解析为 JSON 对象。
      1. 如果读取成功，数据将存储在 data 变量中，并在控制台输出成功消息。
      2. 如果读取失败，捕获错误并在控制台输出错误消息。
  4. 数据处理：
    1. 在程序启动时调用 loadData 函数，并在数据加载完成后进行处理。
    2. 使用 reduce 方法将数据转换为字典 dict，其中键为商品标题，值为商品对象。
    3. 将字典转换为数组 dict_arr，以便进一步处理。
    4. 使用 reduce 方法将 dict_arr 转换为按 location 分组的数组 d，并统计每个 location 的商品数量。
    5. 按 value 值对 d 进行降序排序。
    6. 在控制台输出处理后的数据。
